Global Climate Change Data Explorer

This page showcases two Vega-Lite visualisations for FIT3179 Week 10. The first shows worldwide mammal distributions (from Week 9), and the second displays yearly global land & ocean temperature anomalies from 1850 to today. Both include tooltips, filters, and annotations.

1. Mammal Distribution Map

This map shows the global distribution of mammal habitats by range type (marine, terrestrial, freshwater). Use the legend to highlight a category and hover for details (name, status). The Equal-Earth projection preserves area, improving spatial comparison across regions and coastlines.

2. Global Temperature Anomalies (1850–Present)

The line traces global land–ocean temperature anomalies from 1850 to 2025. Drag the sliders to focus on specific periods. The dashed rules mark the 0 °C baseline and the 1.5 °C threshold. Hover to see exact values and record years.

3. Decadal Average Temperature Anomalies

Bars show the average global temperature anomaly for each decade. Values above the dashed line indicate decades warmer than the long-term baseline.

4. Global Living Planet Index (1970 = 1)

Bars show the average global temperature anomaly for each decade. Values above the dashed line indicate decades warmer than the long-term baseline.
